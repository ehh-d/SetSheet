## Template Format

When requesting workouts, Claude will output in this format:

```
[Workout Category] Day

[Stage Name]
Exercise Name SETSxREPS
Exercise Name SETSxREPS

[Stage Name]
Exercise Name SETSxREPS
Exercise Name SETSxREPS
```

**Example:**

```
Pull Day

[Primer]
Face Pull 2x15
Band Pull Apart 2x15

[Strength]
Lat Pulldown 3x8-10
Bent Over Row 3x10-12
Seated Cable Row 3x10-12

[Capacity]
Bicep Curl 3x10-12
Hammer Curl 3x10-12
```

**Notes:**
- Exercise names must match exactly as listed above
- Equipment variation is optional â€” app will use default or prompt user
- Weight auto-populates from user history
- SETSxREPS format: "3x10" or "3x8-10" for rep ranges
- If no category specified, defaults to "General Workout"
